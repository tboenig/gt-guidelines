📚 Page XML Erweiterungen

Unter PAGE XML Erweiterungen ist nicht im engen Sinn eine Erweiterung des PAGE Schemas zu verstehen.

Eher werden in diesem Teil spezifische Erweiterungen des PAGE-Formates dokumentiert. Ebenfalls werden Hinweise u. a. für die Einbindung von anderen Formaten gegeben.
